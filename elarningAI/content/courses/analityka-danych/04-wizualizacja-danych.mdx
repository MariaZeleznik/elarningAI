## Zasady wizualizacji zanim zaczniesz

Narzędzie (Matplotlib, Plotly) to ostatni wybór. Pierwsze pytania:
1. Co chcę *pokazać*? (trend, rozkład, porównanie, zależność, skład)
2. Dla kogo? (analityk = szczegóły, zarząd = jeden wniosek)
3. Ile kategorii? (<7 kolorów dla czytelności)

| Pytanie | Właściwy wykres |
|---------|----------------|
| Jak zmienia się w czasie? | Liniowy (line chart) |
| Porównanie kategorii | Słupkowy (bar chart) |
| Proporcje całości | Kołowy (pie) - tylko gdy < 5 segmentów |
| Rozkład jednej zmiennej | Histogram, boxplot |
| Zależność X vs Y | Scatter plot |
| Macierz korelacji | Heatmap |

## Matplotlib - podstawy

```python
import matplotlib.pyplot as plt
import numpy as np

# Styl profesjonalny
plt.style.use('seaborn-v0_8-whitegrid')

fig, axes = plt.subplots(2, 2, figsize=(12, 10))
fig.suptitle('Przegląd sprzedaży Q1 2025', fontsize=16, fontweight='bold')

# 1. Trend sprzedaży (line chart)
miesiace = ['Styczeń', 'Luty', 'Marzec']
sprzedaz = [142000, 156000, 178000]
poprzedni_rok = [128000, 141000, 153000]

axes[0,0].plot(miesiace, sprzedaz, 'b-o', linewidth=2, label='2025', markersize=8)
axes[0,0].plot(miesiace, poprzedni_rok, 'r--o', linewidth=2, label='2024', markersize=8, alpha=0.7)
axes[0,0].set_title('Trend sprzedaży')
axes[0,0].set_ylabel('Przychód (PLN)')
axes[0,0].legend()
axes[0,0].yaxis.set_major_formatter(plt.FuncFormatter(lambda x, p: f'{x/1000:.0f}k'))

# 2. Sprzedaż per kategoria (horizontal bar - lepsza gdy długie nazwy)
kategorie = ['Elektronika', 'Odzież', 'Sport', 'Dom i ogród', 'Zabawki']
wartosci = [487000, 234000, 198000, 156000, 89000]
colors = ['#2196F3' if v == max(wartosci) else '#90CAF9' for v in wartosci]

bars = axes[0,1].barh(kategorie, wartosci, color=colors)
axes[0,1].set_title('Sprzedaż per kategoria')
axes[0,1].set_xlabel('Przychód (PLN)')
# Dodaj etykiety wartości
for bar in bars:
    w = bar.get_width()
    axes[0,1].text(w + 5000, bar.get_y() + bar.get_height()/2, 
                   f'{w/1000:.0f}k', va='center', fontsize=9)

# 3. Rozkład zamówień (histogram)
np.random.seed(42)
zamowienia = np.random.lognormal(mean=5, sigma=0.5, size=1000)
axes[1,0].hist(zamowienia, bins=30, color='#4CAF50', edgecolor='white', alpha=0.8)
axes[1,0].axvline(np.median(zamowienia), color='red', linestyle='--', label=f'Mediana: {np.median(zamowienia):.0f} PLN')
axes[1,0].set_title('Rozkład wartości zamówień')
axes[1,0].set_xlabel('Wartość zamówienia (PLN)')
axes[1,0].legend()

# 4. Scatter: przychód vs liczba zamówień per klient
np.random.seed(42)
n = 100
zamowienia_per_klient = np.random.randint(1, 50, n)
przychod_per_klient = zamowienia_per_klient * 150 + np.random.normal(0, 500, n)
kategoria_klienta = np.random.choice(['Gold', 'Silver', 'Basic'], n)

colors_map = {'Gold': '#FFC107', 'Silver': '#9E9E9E', 'Basic': '#2196F3'}
for kat in ['Gold', 'Silver', 'Basic']:
    mask = kategoria_klienta == kat
    axes[1,1].scatter(zamowienia_per_klient[mask], przychod_per_klient[mask],
                      c=colors_map[kat], label=kat, alpha=0.7)
axes[1,1].set_title('Klienci: liczba zamówień vs przychód')
axes[1,1].set_xlabel('Liczba zamówień')
axes[1,1].set_ylabel('Całkowity przychód (PLN)')
axes[1,1].legend()

plt.tight_layout()
plt.savefig('dashboard_q1.png', dpi=150, bbox_inches='tight')
plt.show()
```

## Seaborn - statystyczne wykresy w jednej linii

```python
import seaborn as sns
import pandas as pd

# Generuj dane
np.random.seed(42)
df = pd.DataFrame({
    'dzien_tygodnia': np.tile(['Pon','Wt','Śr','Czw','Pt','Sob','Nd'], 100),
    'kanal': np.random.choice(['Web', 'Mobilna', 'Sklep'], 700),
    'sprzedaz': np.random.lognormal(6, 0.8, 700),
    'region': np.random.choice(['Północ', 'Południe', 'Centrum'], 700),
})

fig, axes = plt.subplots(1, 3, figsize=(16, 5))

# Boxplot - rozkład sprzedaży per kanał
sns.boxplot(data=df, x='kanal', y='sprzedaz', ax=axes[0], palette='Set2')
axes[0].set_title('Rozkład sprzedaży per kanał')
axes[0].set_yscale('log')

# Heatmap - korelacja
numeric_data = df[['sprzedaz']].assign(
    dzien_num=pd.Categorical(df['dzien_tygodnia']).codes,
    kanal_num=pd.Categorical(df['kanal']).codes,
    region_num=pd.Categorical(df['region']).codes,
)
corr = numeric_data.corr()
sns.heatmap(corr, annot=True, cmap='coolwarm', center=0, ax=axes[1], fmt='.2f')
axes[1].set_title('Macierz korelacji')

# Violin plot - rozkład per region
sns.violinplot(data=df, x='region', y='sprzedaz', ax=axes[2], palette='muted')
axes[2].set_title('Rozkład sprzedaży per region')
axes[2].set_yscale('log')

plt.tight_layout()
plt.show()
```

## Zasady dobrego wykresu (do zapamiętania)

1. **Jeden wykres = jeden wniosek** - tytuł to wniosek, nie opis ("Sprzedaż elektroniki rośnie o 23% rok do roku", nie "Wykres sprzedaży elektroniki")
2. **Mniej = więcej** - usuń gridlines, borders, legendy kiedy nie potrzebne
3. **Koloruj ze znaczeniem** - highlight jednej kategorii zamiast tęczy kolorów
4. **Oś Y zaczyna od 0** (dla bar chart) - ucięta oś Y to częsty manipulator
5. **Rozmiar markera/słupka = percepcja** - uważaj na wykres kołowy z 10 kawałkami
